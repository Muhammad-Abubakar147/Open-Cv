# 📸 Phase 5 – Image Filtering with OpenCV

## Welcome to **Phase 5** of Open-Cv

In this phase, I explored the fundamentals of **Image Filtering** using OpenCV. Image filtering is an essential preprocessing technique in computer vision that helps reduce noise, smooth images, and enhance important details.

## 📚 Topics Covered

- 🖼️ Raw Images
- 🌫️ Gaussian Blur
- 🎯 Median Blur
- ✨ Sharpening Filters
