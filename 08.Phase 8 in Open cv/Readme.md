# 👁️ Object & Face Detection using OpenCV

This phase demonstrates how to detect faces and objects in images and videos using OpenCV.

## 📂 Files Included

- **Face Detection**
  - Detects human faces using Haar Cascade Classifier.

- **Object Detection**
  - Detects objects from images or live webcam feed using OpenCV.

## 🛠️ Technologies Used

- Python
- OpenCV (`cv2`)
- Haar Cascade Classifiers
- NumPy
